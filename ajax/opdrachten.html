Opdrachten week 4

Deze week gaan we oefenen met Asynchronous JavaScript and XML, oftewel AJAX. We hebben een aantal webservices klaargezet waar jullie mee kunnen oefenen. Je mag voor alle opdrachten jQuery gebruiken om het jezelf makkelijk te maken.

Quote

Op deze URL: http://network.aii.avans.nl/quote staat een webservice die een willekeurige quote teruggeeft. Deze site is speciaal ingesteld zodat je er vanaf elk domein een Ajax request naar mag doen.

Maak een site waarbij je een willekeurige quote laat zien geladen uit bovenstaande URL. Gebruik Ajax en JavaScript om dat voor elkaar te krijgen

Als je voor een pixel-perfecte kopie van de screenshot gaat mag je gebruik maken van onze HTML en CSS

Hint: Met de jQuery load() functie kan je jezelf het er heel makkelijk vanaf maken

Weer

Met onze weer webservice http://network.aii.avans.nl/weather?location=Eindhoven kan je dynamisch weergegevens ophalen over een bepaalde stad. Die stad moet je als 'location' parameter meegeven in de URL.

Maak een kleine webapplicatie waarop de gebruiker een stad kan invullen en op een knop kan klikken. Jouw JavaScript haalt dan de huidige temperatuur op in die stad en toont die op de webpagina.

(Data is afkomstig van http://api.openweathermap.org/data/2.5/weather?q=Eindhoven maar die service is helaas niet ingesteld dat je er met Ajax bij mag.)

HTML & CSS

Chat

Met de komst van Ajax hoeven we niet meer Flash of Java-applets te gebruiken om chatrooms te implementeren. Alles is mogelijk met alleen maar HTML en JavaScript.

De moeilijke opdracht deze week is om een online chatpagina te maken. Laat bij het opstarten meteen de laatste berichten in de chat zien. Nadat de gebruiker een berichtje verstuurd ververs je de laatste chatberichten

Als je een request doet naar http://network.aii.avans.nl/chat krijg je een lijst van de laatste chatberichten in onze Avans chatroom. Je kan zelf ook een berichtje plaatsen door een POST request te doen naar dezelfde URL met het volgende JSON bericht: {"nickname": "Naam", "message": "Bericht"}
