Een eerste programma

JavaScript schrijven is de makkelijkste programmeertaal om te installeren, we hebben namelijk geen speciale IDE's of compilers nodig. Het enige wat je nodig hebt is: 1. een browser en 2. een teksteditor. En je hebt inmiddels vast een favoriete versie van beide.

Een belangrijke tool die je gaat gebruiken zijn de ontwikkelhulpprogramma's die in elke browser zitten (onderaan deze pagina staat hoe je die opent). In het Console tabje zie je alle fouten die in je Javascript zitten en de output van de console.log() functie. We kunnen hier ook rechtstreeks JavaScript typen.

Probeer de volgende stukjes JavaScript uit in de console en kijk wat ze doen:

  • alert("Hallo Wereld");
  • console.log("Hallo Wereld");
  • Math.random();
  • window.confirm("Weet u het zeker?");
  • window.prompt("Wat is je naam?");
  • window.location.href = "https://www.youtube.com/watch?v=dQw4w9WgXcQ";

JavaScript is net als PHP een interpreted taal, wat wil zeggen dat je code niet eerst gecompileerd hoeft te worden zoals bij Java of C#. Als er ergens een syntax fout zit wordt alle code tot die fout gewoon uitgevoerd. In de Console krijg je te zien wat de fout is als je script niet goed runt.

Script tag

Als we JavaScript willen toevoegen aan een webpagina hebben we de <script> tag nodig. Tussen de open en sluit tag zet je vervolgens alle JavaScript neer die je maar wilt.

<!DOCTYPE html> <html> <head> </head> <body> <script> alert("Hallo JavaScript!"); </script> </body> </html>

Maak een pagina die jouw naam in een alert() dialoogje zet als je de pagina opent

Net zoals CSS kan je JavaScript in een extern bestand plaatsen. Dat doe je dan met het src attribuut van de script tag:

<script src="mijnjavascript.js"></script>

Als je meer wil doen dan een paar regels JavaScript kan je het beste deze aanpak gebruiken.

Attributen

Een andere manier om JavaScript uit te voeren is via speciale eventhandler attributen. Met het onclick kan je bijvoorbeeld code uitvoeren als er op een knopje geklikt wordt:

<button onclick="alert('klik!');">Knopje</button>

Naast onclick zijn er nog vele andere event handlers. Volgende week gaan we zien dat je ook vanuit JavaScript zelf event handlers kan toevoegen aan een pagina. En dat is eigenlijk ook de betere manier om het te doen. Net zoals je CSS liefst zoveel mogelijk in een apart bestand wil stoppen wil je dat met JavaScript ook. Zo behoud je de separation of concerns tussen JavaScript, HTML en CSS.
