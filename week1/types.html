Variabelen & types

JavaScript is een dynamische programmeertaal, dat wil zeggen dat variabele geen vast type hebben. Net als PHP is het prima als je in dezelfde variabele eerst een boolean, dan een getal en dan een string in stopt. Een variabele declareer je door var voor de naam te zetten:

var a = "hallo";

Misschien ben je gewend om var weg te laten, we zullen later zien bij het hoofdstuk "Scope" zien dat het dan heel wat anders betekend (ondanks dat het wel vaak hetzelfde werkt). Zorg er dus voor dat je gewoon netjes var voor al je lokale variabele zet

Types

JavaScript kent vijf primitieve types:

  • undefined
  • null
  • Number (2, -1 of 3.14)
  • Boolean (true, false)
  • String ("Hallo")

Alle andere waardes zijn objecten
