JavaScript

Welkom bij WEBS3, het JavaScript vak! We gaan er bij dit vak van uit dat je al prima kan programmeren in talen als PHP, Java en C#. We gaan dus vooral kijken waar JavaScript anders is, en dat is eigenlijk best wel veel.

Wat is JavaScript?

JavaScript is een multi-paradigma, duck-typed, objectgeorienteerde, prototype gebaseerde, imperatieve, functionele programmeertaal. Nee wacht, laat ik anders beginnen. JavaScript is die programmeertaal die in je browser ervoor zorgt dat je webpagina's ineractief en slim zijn. Behold, the power of JavaScript:

Naast flauwe spelletjes kan je JavaScript voor nog veel meer gebruiken. Moderne webapplicaties zijn grotendeels JavaScript met nog een beetje HTML en CSS eroverheen gesprenkeld for good measure.

Wat is JavaScript niet?

De naam doet vermoeden dat JavaScript een afgeleide is van Java. Maar niets is minder waar. Javascript lijkt net zoveel op Java als PHP op C# lijkt, niet zoveel dus... Maar waarom dan die gekke naam?

De eerste versies van Javascript gingen door Livescript door het leven in de goede oude tijd van 1995. Om de illusie te wekken van een 'hot new programming language' hebben ze de naam veranderd in JavaScript, Java was (en is) immers een populaire programmeertaal. Niets meer dan een simpel marketingtruucje dus.

De taal zelf heet eigenlijk ECMAScript, en JavaScript is alleen de implementatie ervan in de browser. Maar omdat ECMAScript vies klinkt en lastig is uit te spreken hoor je weinig mensen die term gebruiken.
